# Mental Health Awareness — Local Preview

This repository contains a simple accessible homepage for mental health awareness.

Files:
- `index.html` — Main homepage
- `styles.css` — Styling for the homepage

Preview locally

Option 1 — Open in your browser (quick):

Double-click `index.html` in File Explorer or open it with your browser.

Option 2 — Run a simple local web server (recommended):

Open PowerShell in this folder and run:

```powershell
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```

If you don't have Python, you can install it from https://python.org or use an editor that serves files.
